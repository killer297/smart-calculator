# smart-calculator

**This is my first application which is dedicated to mobile software.**

#📱 Smart Calculator App

A modern, intelligent calculator mobile application designed to go beyond basic arithmetic. The Smart Calculator combines powerful computation features with a clean UI and smart capabilities like history tracking, expression parsing, and advanced calculations.

🚀 Features
🧮 Basic Calculations
<br>
Addition, subtraction, multiplication, and division
<br><br><br>
📊 Advanced Functions
<br>
Trigonometric functions (sin, cos, tan)
Logarithms and exponentials
Square root and power operations
<br><br><br>
🧠 Smart Input Parsing
<br><br>
Supports full mathematical expressions (e.g., 5 + 3 * (2^2))
📜 Calculation History
View, reuse, or delete previous calculations
💡 Real-Time Results
Instant evaluation as you type
<br><br>
🌙 Dark Mode Support
Eye-friendly UI for night usage
<br>
🔄 Responsive Design
Optimized for different mobile screen sizes
<br>
🛠️ Tech Stack
Language: Dart / JavaScript / Kotlin / Java
State Management: Provider / Redux / Bloc (depending on framework)
